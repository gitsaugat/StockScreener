Introduction

  • The stock market generates vast amounts of financial and price data every second.
  • Investors often struggle to interpret these data efficiently.
  • This project leverages AI to simplify stock screening and provide actionable investment insights.
  • The system integrates real-time data, machine learning, and AI-based interpretation into one platform.

Project Overview

  • A Flask-based web application for analyzing stock performance.
  • Fetches real-time data from yFinance and performs AI-driven analysis using Gemini API.
  • Stores results and user-selected stocks in a PostgreSQL database.
  • Provides structured financial reports including risk, valuation, and recommendations.

Project Objectives

  • Build a Flask web app for interactive stock analysis.
  • Integrate yFinance to fetch live stock and financial data.
  • Use Gemini API for natural language financial analysis.
  • Implement PostgreSQL for persistent data storage.
  • Generate a complete financial report with:
    • Market Sentiment (Bullish/Bearish/Neutral)
    • Key Strengths & Concerns
    • Valuation & Risk Analysis
    • Investment Recommendation
